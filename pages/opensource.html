Open Source Tools

SURROUND proudly supports a range of Open Source tools that assist organisations with a number of Sematnic Web-related tasks.


RDFlib


RDFlib is a family of Python tools for RDF manipulation.

It contains a number of Stores, parsers/serializers & Graph modules that allow most of the expeceted RDF tasks such as file reading, SPARQL querying and so on to be performed in relateively few lines of Python code.

pyLODE is an OWL ontology documentation tool using Python and templating, based on LODE.

pySHACL is a family of Python tools for RDF manipulation.

It contains a number of Stores, parsers/serializers & Graph modules that allow most of the expeceted RDF tasks such as file reading, SPARQL querying and so on to be performed in relateively few lines of Python code.

Cheka is a profile hierarchy-based RDF graph validation tool written in Python

It validates a data graph against a set of SHACL shape graphs that it extracts from a hierarchy of Profiles (Standards/Specifications and/or profiles of them). It uses conformance claims in the data graph to a Profile to collate and then use all the validator SHACL files within the hierarchy of other Profiles and Standards to which that Profile profiles.
